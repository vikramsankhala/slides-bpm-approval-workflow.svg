## Quick Start

1. **Create your slides** as SVG files in the `slides/` folder
2. **Update the slides array** in `index.html`:
   ```javascript
   const slides = [
     'slides/slide1.svg',
     'slides/slide2.svg',
     'slides/slide3.svg'
   ];
   ```
3. **Open `index.html`** in a web browser (or serve via web server)
